PAD REST API

A very lightweight Puzzles and Dragons API.

Available Endpoints

The following endpoints are available:

  • /api/status
  • /api/monsters
  • /api/monsters/{id}

Example Responses

/api/status

    {
        "status": "UPDATING",
        "timestamp": "2018-08-10T23:26:21.456+0000",
        "count": 0
    }
